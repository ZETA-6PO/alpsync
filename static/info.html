Introducing AlpSync: Eco-Friendly File Sharing
AlpSync, developed for the ICT171 course at Murdoch University, offers a streamlined online file-sharing service.
Inspired by platforms like WeTransfer, AlpSync simplifies the process of uploading files and obtaining shareable links.

Key Features:
1. Effortless Uploads: Drag and drop files onto AlpSync's interface or select them from your directories.
2. Instant Shareable Links: AlpSync generates unique links for each file upon upload, allowing instant sharing.
3. User-Friendly Interface: Enjoy a simple and intuitive interface for seamless navigation.
4. Carbon Footprint Tracking: AlpSync displays the carbon cost of file storage, promoting eco-conscious file management.
5. Secure and Reliable: Your files are protected with robust security measures.
6. Apache License v2: AlpSync is released under the Apache License v2, fostering collaboration and innovation.

Experience modern, eco-friendly file sharing with AlpSync.
